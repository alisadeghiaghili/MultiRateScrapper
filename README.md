
# Multi Rate Scraper

This Python code scrapes several rates, rate changes and change amounts from tgju.org and then write it in the DataBase.
Theses rates includes:
- USD/Rls
- Iran stock exchange index
- Gold(oz)/Rls
- BTC/Rls
- USDT/RLS
- Gold(g)/Rls
- Brent Oil/Rls
- Imami gold coin/Rls
